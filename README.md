## What is a SIEM?
A SIEM (Security Information and Event Management) platform collects, aggregates, and analyzes logs from multiple systems to detect suspicious activity.

## Core SIEM Functions
- Log aggregation
- Correlation rules
- Alerting
- Dashboards
- Incident investigation
- Compliance reporting

## Common Log Sources
- Windows Event Logs
- Firewalls
- IDS/IPS
- Authentication logs
- Web server logs
- Endpoint telemetry

## SIEM Workflow
1. Collect logs
2. Normalize data
3. Correlate events
4. Generate alerts
5. Investigate incidents

## Detection Example
Example:
- Multiple failed logins
- Followed by a successful login
- From a suspicious IP

This could trigger a brute-force detection rule.
